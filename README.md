# KfWL
KfWL - Keys for White List 
Сейчас в России действуют белые списки - доступны ГосУслуги, Яндекс, ВК. Остальные сервисы недоступны.


Мой проект помогает восстановить стабильное соединение с ресурсами, к которым затруднен из за некорректной настройки сетевых маршрутов или фильтров оператора. Это позволяет вернуть интернет в его исходное, рабочее состояние.


Я не создаю и не арендую собственные конфиги, я сортирую и выкладываю лучшие конфиги проверенные на пинг.

Авторы подписок из которых я сортирую рабочие обходы: NowMeow Whitelist, WL RUS #1, WL RUS Selected, igareck Cidr, AvenCores Goida #26, zieng2 vless universal и lite, a9fm EtoNeYaProject (Базовая и тест), LowiKLive (Бесплатная подписка), RosTunnel MEGA, GoodBywWL, xcom024, newsigot, airlinkvpn, UWB (Общая подписка и с сайта), vpn abroad, OBSpiskov, communitycatler.


Я не гарантирую полную работоспособность подписки, у всех разные операторы.


KfWL.txt Файл где лежит список отборных конфигураций. Актуальная ссылка https://raw.githubusercontent.com/prominbro/KfWL/refs/heads/main/KfWL.txt Скопируйте и вставьте ее в свой клиент.



 Disclaimer. Важное уточнение: Данный репозиторий не является инструментом для обхода блокировок. Я не предоставляю способы обхода, а лишь демонстрирую возможности настройки безопасного и стабильного интернет-соединения в образовательных целях. Все данные собраны из открытых источников для аудита безопасности.
